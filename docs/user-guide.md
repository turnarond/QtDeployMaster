# DeviceForge 用户手册

## 快速入门

1. 从 [Releases](../../releases) 下载 `DeviceForge-v2.1.0-win64.zip`
2. 解压到任意目录
3. 运行 `DeviceForge.exe`

> 需要 [VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)（已安装 VS2022 可跳过）

## 功能模块

### 文件部署

批量上传文件/文件夹到多台 FTP 设备。

1. 在设备总线添加目标设备（点 "+" 按钮）
2. 切换到「文件部署」Tab
3. 设置远程路径（如 `/app`）
4. 勾选 **FTPS 加密传输**（推荐）
5. 点击「+ 添加文件」选择本地文件
6. 点击「开始部署」

### 批量命令

通过 Telnet 批量执行 Shell 命令。

1. 在设备总线添加目标设备
2. 切换到「批量命令」Tab
3. 输入要执行的命令（每行一条）
4. 设置超时时间
5. 点击「执行」
6. 查看结果表格和输出详情

### Modbus 测试

批量读写 Modbus TCP 寄存器。

1. 在设备总线添加 Modbus 设备
2. 切换到「MODBUS 测试」Tab
3. 选择寄存器类型（Holding/Input/Coils/Discrete）
4. 设置起始地址和数量
5. 点击「读取」或「自动刷新」

### WebSocket 通信

内建 Server 和 Client 双模式。

- **Server 模式**：在本机启动 WebSocket 服务器，默认监听 127.0.0.1
- **Client 模式**：连接到远端 WebSocket 服务器
- 支持 SUBSCRIBE/PUBLISH 主题订阅

### 网络调试（中继 + 录制回放 + 组播）

在不影响生产数据流运行的前提下，串接在数据生产者与真实消费者之间，原样双向转发流量并旁路抓包，还可录制流量供日后回放。新增组播（Multicast）支持——以额外订阅者身份加入组播组抓收数据并回灌。

**中继抓包：**

1. 切换到「网络调试」Tab
2. 选择协议（TCP / UDP / **Multicast**）
3. 设置**监听地址**和端口（TCP/UDP 时工具在此等待生产者连接；组播时此参数忽略）
4. 设置**上游地址**和端口（TCP/UDP 时指真实消费者；**组播时指组播组地址和端口**，如 `239.1.2.3:5000`）
5. **组播时额外选择网卡**（多网卡机器必须选对网卡，否则收不到组播数据）
6. （可选）勾选「录制到文件」并选择 `.nrec` 保存路径
7. 点击「▶ 启动」
7. 让数据生产者改为连接工具的监听地址；工具会把流量原封不动转发到上游消费者
8. 在「十六进制视图」实时查看上/下行数据（上行青绿、下行琥珀），「会话列表」查看各连接字节数
9. 点击「⬇ 导出」保存当前 Hex 视图快照，或「■ 停止」结束并保存录制

**回放：**

1. 先「■ 停止」中继（回放与中继互斥）
2. 在「回放」区选择之前录制的 `.nrec` 文件
3. 设置**消费者**地址端口（回放数据发往此处，模拟生产者）
4. 选择速率（原速 = 还原原始时间间隔 / 尽快）
5. 点击「▶ 回放」，进度条显示已回放条数；可「⏸ 暂停」/「■ 停止」

> **说明**：回放只重放录制中的**上行**数据（生产者→消费者方向）到消费者；下行数据保留在文件中但当前不回放。组播回放默认灌回录制时的原组播组地址/端口，可在回放区覆盖。
>
> **安全提示**：监听地址默认为 `127.0.0.1`；填写非回环地址会输出安全警告
>
> **安全提示**：监听地址默认为 `127.0.0.1`；填写非回环地址会输出安全警告（中继无客户端鉴权，暴露到网络需环境可信）。`.nrec` 录制与导出文件可能含明文凭证/敏感数据，导出前会二次确认，文件权限限制为仅所有者可读写。

### 远端预览

浏览和下载远程 FTP 设备上的文件。

1. 选择协议（FTP）
2. 选择目标设备
3. 输入远程路径
4. 点击「刷新」
5. 双击目录进入，双击文件下载
6. 或选中文件点击「下载」

## 安全功能

- **FTPS 加密**：文件部署 Tab 中勾选
- **Telnet 警告**：首次使用时弹出明文传输风险提示
- **WebSocket 认证**：可配置 Token 认证（URL 参数 `?token=xxx`）
- **网络中继**：监听地址默认 127.0.0.1，非回环绑定输出安全警告；录制/导出文件二次确认 + 仅所有者可读写
- **密码安全擦除**：断开连接后自动擦除内存中的密码
